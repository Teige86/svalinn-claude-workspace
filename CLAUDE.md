# Instruks for Claude

Dette repository-et inneholder dokumenter, PDF-er, tekstuttrekk, sammendrag og arbeidsfiler.

## Viktig regel for PDF-er

Claude skal ikke lese original-PDF-er direkte med mindre det er nødvendig.

Arbeidsrekkefølge:

1. Les først `pdf-index.md`
2. Les deretter relevante filer i `03-sammendrag/`
3. Les filer i `02-tekstuttrekk/` hvis mer detalj trengs
4. Les original-PDF i `01-originaler/` kun hvis:
   - eksakt ordlyd må kontrolleres
   - sidereferanse må bekreftes
   - tabell eller figur ikke finnes i tekstuttrekket
   - bruker eksplisitt ber om kontroll mot originalen

Når original-PDF brukes, skal Claude kun lese relevante sider eller avsnitt, ikke hele dokumentet.

## Prioritering ved anbud/tilbud

Ved arbeid med anbudsdokumenter skal Claude prioritere:

- frister
- krav
- prisskjema
- forbehold
- risiko
- kontraktsvilkår
- leveranseomfang
- avvik mellom dokumenter

Claude skal alltid si fra hvis svaret bygger på sammendrag og ikke er kontrollert mot original-PDF.
